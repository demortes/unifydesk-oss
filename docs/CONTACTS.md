## Reporting

### Bug Reports
1. Check [existing issues](https://github.com/Demortes/unifydesk/issues)
2. Use the bug report template
3. Include all required information
4. Do not include sensitive data

### Feature Requests
1. Check if already requested
2. Use the feature request template
3. Describe the problem and proposed solution
4. Indicate which platforms are affected

### Security Vulnerabilities
1. **DO NOT** create a public issue
2. Email: unifydesk@demortes.com
3. Use GitHub's private vulnerability reporting
4. Wait for response before public disclosure
